# Astrology Site 
This project is a grate web page to get your daily horoscope from. It uses the [Horoscope API from astrology.com](http://ohmanda.com/api/horoscope/)

## Tech Used
- JavaScript
- Node.js
- Bootstrap
- HTML/CSS
- Pug

## Setup, Install & Run
```
npm install
```

## References
- [API](http://ohmanda.com/api/horoscope/)
